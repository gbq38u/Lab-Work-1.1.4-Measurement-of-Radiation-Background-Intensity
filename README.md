# Lab-Work-1.1.4-Measurement-of-Radiation-Background-Intensity
Эксперимент посвящён статистическому исследованию процессов радиоактивного распада с использованием счётчика Гейгера–Мюллера. Измеряется число актов регистрации частиц за фиксированные интервалы времени, строятся гистограммы распределений, которые сравниваются с теоретическими распределениями Пуассона и Гаусса.
